# Reglamento Simulado de Becas RD (MVP)

Este documento resume reglas base usadas para el sistema de orientacion del proyecto academico.

## Tipos de beca

- Beca nacional: cobertura parcial o total para programas en universidades de Republica Dominicana.
- Beca internacional: apoyo para estudios fuera del pais, priorizando programas de maestria y doctorado.

## Requisitos generales

- Formulario de postulacion completo y firmado.
- Record academico oficial.
- Copia de cedula o pasaporte vigente.
- Carta de motivacion y plan de estudios.
- Evidencia socioeconomica del hogar.

## Requisitos adicionales para beca internacional

- Carta de admision o pre-admision de universidad destino.
- Evidencia de idioma (ingles u otro segun programa).
- Presupuesto estimado anual.

## Criterios de evaluacion (orientativos)

- Merito academico.
- Calidad y consistencia documental.
- Necesidad economica.
- Alineacion del programa con prioridades de desarrollo nacional.

## Nota

La decision final siempre corresponde al comite oficial de becas.
