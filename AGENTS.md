# AGENTS.md

## Instrucciones del repositorio

- Revisa `PLANS.md` al inicio de cada tarea para entender el avance actual.
- Actualiza `PLANS.md` cuando cierres una tarea o cambies el estado de una fase.
- Usa `docs/portal_ciudadano_plan.md` como fuente de verdad del plan funcional y de producto del portal ciudadano.
- Si una decision nueva cambia alcance, arquitectura o prioridades, refleja el cambio primero en `docs/portal_ciudadano_plan.md` y luego en `PLANS.md`.
- No marques tareas como completadas sin haber validado el resultado con pruebas o evidencia suficiente para esa fase.
